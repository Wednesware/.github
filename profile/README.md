# Concept Roadmap:
* Oxygen - animations, gradients
* Nitro (name may likely change) - Smaller, faster Nitrogen as a library
* Krypton - encryption, save files, encoding
* Sodium - plugin marketplace
* Helium mod loader capabilities
* Iodine - user input, choices
* Hydrogen - Web-based applications
* Unidep/Getdep - PyPI package with a 'getdep' command that runs Nitrogen NitroDep and Sodium PluginDep
